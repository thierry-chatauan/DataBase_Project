About the Project

This project is a web application designed to manage job records for a manufacturing or production environment. It allows users to create, edit, and delete job entries, as well as track the start and end times of each job. The application is built using Flask, a lightweight web framework for Python, and utilizes HTML and CSS for the front-end design.

Overall, the project provides a user-friendly interface for efficiently managing job records and keeps track of production processes.
